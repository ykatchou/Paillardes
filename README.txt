Done by Yoann Katchourine (2010-2011)

Il s'agit d'un bréviaire de chanson paillardes pour Android.
Assez simple mais robuste et efficace.

Le dépot comportent plusieurs projets :
 - Paillarde : L'application android (projet eclipse)
 - SongsManager : Un projet python permettant de manipuler les données de l'application
